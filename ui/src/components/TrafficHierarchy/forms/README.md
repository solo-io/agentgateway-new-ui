# Traffic Forms

Manual TypeScript/JSON Schema definitions for the Traffic hierarchy page.

## Files

- **bindForm.ts** — Schema for `LocalBind` (port bindings)
- **listenerForm.ts** — Schema for `LocalListener` (protocol listeners)
- **routeForm.ts** — Schema for `LocalRoute` (HTTP/TCP routes)
- **backendForm.ts** — Schema for `LocalRouteBackend` (service/host/MCP backends)
- **index.ts** — Exports all forms

For schema authoring patterns, UI schema options, and maintenance guidance see [traffic-hierarchy-ai.md](../traffic-hierarchy-ai.md).
